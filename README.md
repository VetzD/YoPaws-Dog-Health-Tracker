# YoPaws Dog Health Tracker

A simple web app designed to help dog owners track their dog's daily health information in one place.

Built for real-life dog handling situations where quick logging matters.

## Features

• Track vomiting, stool condition, appetite, and behaviour  
• Record medications and dosage times  
• Store dog profile details  
• Add photos of your dog  
• Works in browser and as a Progressive Web App (PWA)  
• Local storage so data stays on your device  

## Purpose

This tool was originally created to help track health patterns for dogs dealing with illness, medication changes, or behavioural stress.

It is especially useful for:

• Assistance dog handlers  
• Trainers monitoring dog health  
• Owners tracking symptoms over time
